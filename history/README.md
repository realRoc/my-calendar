# history/

Past actions and feedback, organized by year:

```
history/
├── 2026/
│   ├── 2026-05-10__mothers-day__mom.md
│   ├── 2026-09-12__mom-birthday__mom.md
│   └── ...
└── 2027/
    └── ...
```

Filename convention: `<YYYY-MM-DD>__<holiday-id>__<person-id>.md` (double-underscore separated).

Schema in [`../AGENTS.md`](../AGENTS.md#history-文件). Created automatically when you tell Claude:

> "记一下今天送了妈妈 580 块剧院票"

Contents of this directory (other than this README) are gitignored.

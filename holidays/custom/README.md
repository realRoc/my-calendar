# holidays/custom/

Your own holidays — typically family birthdays, anniversaries, private dates.

Schema and date-rule types in [`../../AGENTS.md`](../../AGENTS.md#holiday-文件). Created
automatically when you tell Claude:

> "加一个节日：奶奶生日是农历七月初九"

Contents of this directory (other than this README) are gitignored.

Predefined holidays (Chinese legal + Western common) live in `../cn-legal/` and `../western/` —
those are tracked by git and shipped with the project.

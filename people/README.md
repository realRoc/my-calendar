# people/

Per-person profiles (mom, dad, friends, ...).

Each file is `<person-id>.md` with YAML frontmatter — see [`../AGENTS.md`](../AGENTS.md#person-文件)
for the schema. Files are created automatically when you tell Claude:

> "新增一个人物：妈妈，9 月 12 日生日，喜欢吃鱼"

The contents of this directory (other than this README) are gitignored, so your
personal profiles never end up in the public repo.
